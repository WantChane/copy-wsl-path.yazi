# copy-wsl-path.yazi

## Usage

```toml
[[mgr.prepend_keymap]]
on = ["c", "w"]
run = "plugin copy-wsl-path win"
desc = "Copy the file path in windows"
```

```toml
[[mgr.prepend_keymap]]
on = ["c", "w"]
run = "plugin copy-wsl-path wsl"
desc = "Copy the file path in wsl"
```
